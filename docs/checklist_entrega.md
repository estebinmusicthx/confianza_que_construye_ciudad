# Checklist de entrega DataJam

- [ ] Repositorio público o con permisos habilitados.
- [ ] README.md con problema, fuentes, metodología e instrucciones.
- [ ] Notebook reproducible en `/notebooks`.
- [ ] Script equivalente en `/scripts`.
- [ ] `requirements.txt`.
- [ ] Dashboard HTML o enlace a visualización.
- [ ] CSV limpios en `/outputs/clean`.
- [ ] Nota técnica de integración en `/docs`.
- [ ] Formulario de caracterización diligenciado.
- [ ] Fuentes y rutas citadas.
- [ ] Pitch de 3 minutos.
- [ ] Verificar que el dashboard abra correctamente.
- [ ] Enviar correo con link del repositorio y visualización.
