# Argumentario para preguntas del jurado

## ¿La hipótesis es verificable?
Sí. Se verifica si existen asociaciones territoriales entre ITCC, cumplimiento predial, morosidad y variables de proyectos/participación. No se verifica comportamiento individual.

## ¿Hay causalidad?
No se afirma causalidad. Se trabaja con asociaciones y condiciones territoriales.

## ¿Por qué no cruzan aporte voluntario por localidad?
Porque la fuente pública de aporte voluntario predial no incluye localidad. Hacerlo sería metodológicamente incorrecto. Por eso el aporte se analiza por año/estrato y la estrategia territorial se construye con cumplimiento predial, EM2021 y proyectos.

## ¿Cuál es la innovación?
No es el tablero. Es un modelo de decisión pública reproducible: ITCC + cuadrantes + recomendaciones diferenciadas.

## ¿Qué valor público genera?
Permite pasar de campañas genéricas de recaudo a estrategias territoriales de transparencia, pedagogía y reconocimiento ciudadano.

## ¿Es reproducible?
Sí. Tiene notebook, script, fuentes, rutas, requirements, outputs y metodología documentada.
